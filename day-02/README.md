Day 02 - DNA Base Counter

Description:

This program reads DNA sequences from a FASTA file and reports:

* Sequence length
* GC content
* AT content
* Number of A, T, G and C bases

Example:

Sequence: BRCA1

Length: 120 bp

Base Counts
-----------
A: 25
T: 31
G: 34
C: 30

GC Content: 53.33%
AT Content: 46.67%

Dictionaries
Loops
Functions
FASTA parsing
String methods
