# Day 02 - DNA Base Counter

Description:

This project reads DNA sequences from a FASTA file and analyzes their nucleotide composition.

For each sequence, the program calculates the sequence length, GC content, AT content, and the number of occurrences of each DNA nucleotide (A, T, G, and C).

This program reads DNA sequences from a FASTA file and reports:

* Sequence length
* GC content
* AT content
* Number of A, T, G and C bases
* Process multiple DNA sequences

Features:

* Read DNA sequences from a FASTA file
* Calculate sequence length
* Count DNA nucleotides (A, T, G, and C)
* Calculate GC content
* Calculate AT content
* Process multiple DNA sequences

Input (FASTA):

Sequence: BRCA1

Length: 120 bp

Base Counts:

A: 25
T: 31
G: 34
C: 30

GC Content: 53.33%
AT Content: 46.67%
